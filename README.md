Finance Management System (MERN Stack)

Developed a full-stack personal finance management web app using MongoDB, Express, React (Vite), and Node.js.

Implemented JWT authentication for secure user login, registration, and profile management.

Built transaction management module to add, edit, delete, and categorize income/expenses.

Designed dashboard with charts (Chart.js/Recharts) for expense trends, income vs. expenses, and category-wise analysis.

Integrated budget planning with alerts for overspending.

Added search, filter, and export (CSV/PDF) features for better financial tracking.
